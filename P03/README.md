# **P03: Rèplica web**

Després de superar la primera fase de supervivència tècnica a EverPia, l’equip de la consultora ha rebut un nou encàrrec d’un client exigent: **replicar en local una web corporativa real feta amb WordPress**, per analitzar-ne l’estructura, entendre’n l’arquitectura de continguts i aprendre bones pràctiques de desenvolupament i manteniment web professional.

Aquest repte s’emmarca dins l’escenari de **“Sobreviure en una empresa IT”**, on l’objectiu no és només fer funcionar els sistemes, sinó fer-ho amb criteri, rigor tècnic i capacitat d’adaptació a webs complexes i ben dissenyades. En aquest cas, el client vol una **còpia de la web [https://davidperalvarez.com/](https://davidperalvarez.com/)**, un lloc que combina claredat, estructura SEO, disseny net i continguts professionals.

L’objectiu final és **aprendre a replicar fidelment una web existent**, tant a nivell de pàgines i enllaços com d’aparença i organització interna, tot treballant des d’un entorn local de proves amb **WP Local**.

**Instruccions**

El producte **P03: Rèplica web** consisteix en la **creació d’una còpia en local de la web [davidperalvarez.com](https://davidperalvarez.com)** utilitzant **WordPress**.

Cada consultor, de forma individual haurà d’instal·lar un nou lloc al seu WP Local i construir manualment totes les pàgines que apareixen al mapa del lloc XML (sitemap) del web original.

No es tracta només de copiar estèticament, sinó d’**entendre com està construïda**, quina jerarquia de continguts té i com s’organitzen els menús, les categories, els estils i la navegació.

Aquesta pràctica permetrà:

* Analitzar una web professional feta amb WordPress.  
* Reproduir-ne l’estructura amb eines pròpies.  
* Entendre la importància de la coherència visual, la jerarquia i l’experiència d’usuari (UX).  
* Preparar els alumnes per a futurs projectes on hauran de crear o mantenir webs reals per a clients.

Instruccions de realització

1. **Entorn de treball**  
   * Utilitzeu **WP Local** instal·lat al PC de classe.

Creeu una nova web anomenada:

 `replica_[nom]`

*  (per exemple: `replica_Cristian`).

2. **Investigació prèvia**

   * Visiteu la web original i exploreu-ne totes les seccions.  
   * Analitzeu:  
     * La navegació principal (menús, submenús).  
     * L’estructura de continguts (textos, imatges, vídeos, seccions).  
     * L’estil visual (colors, tipografia, distribució).  
   * Investiga amb quins plugins o themes està feta la web, fes servir les eines:  
     * Per crear més blocs, us recomanem molt que us instal·leu el plugin **Kadence Blocks – Gutenberg Blocks for Page Builder Features**  
     * [https://www.wpthemedetector.com/](https://www.wpthemedetector.com/)   
     * Pots esbrinar el color exacte de la web a través de la web [https://imagecolorpicker.com/ca](https://imagecolorpicker.com/ca)  
     * Pots esbrinar la font utilitzada en aquesta web: [https://www.myfonts.com/es](https://www.myfonts.com/es)   
   * Podeu utilitzar el sitemap XML per veure totes les URL existents:  
      [https://davidperalvarez.com/page-sitemap.xml](https://davidperalvarez.com/page-sitemap.xml)

3. **Creació de pàgines**

   * Creeu totes les pàgines principals del web **amb els seus continguts** com:  
     * **Inici**  
     * **Sobre mí**  
     * **Academia básica**  
     * **Mantenimiento para academias online**  
     * **Consultoría para academias online**  
     * **Proyectos que he realizado**  
     * **Descarga**  
     * **Contactar**  
     * **Política de privacidad, cookies, condiciones de uso**, etc.

   * Reproduïu els continguts amb el text copiat de la web o fent servir  (Lorem Ipsum), però mantenint els **títols, seccions i estructura visual** originals.

4. **Configuració del lloc**

   * Definiu una **pàgina d’inici estàtica**.  
   * Configureu el menú principal i els submenús.  
   * Activeu i proveu diferents temes fins a trobar-ne un que s’ajusti al disseny original.  
   * Ajusteu els enllaços permanents i els paràmetres generals (idioma, zona horària, nom del lloc…)
